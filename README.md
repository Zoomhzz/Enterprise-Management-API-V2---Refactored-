# Enterprise-Management-API-V2---Refactored-
API REST corporativa desenvolvida com Java 17 e Spring Boot 3. O projeto foca em segurança e escalabilidade, utilizando o padrão DTO para proteção de dados sensíveis e BCrypt para criptografia de senhas. Implementa Bean Validation para integridade das informações e Stream API para um código funcional, limpo e eficiente. 🚀
